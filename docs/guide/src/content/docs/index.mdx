Net Worth Navigator is a local net worth projection tool that models your financial trajectory across multiple scenarios.

Net Worth Navigator projects your household net worth forward
**year by year**, modeling compound growth, income, spending, taxes,
loans, and life events. It answers questions like:

{/* deploy-timestamp: 2026-07-19T23:42:00 */}

- _Are we on track for retirement?_
- _What happens if one of us retires earlier?_
- _How long does the portfolio last at this spending level?_
- _What if we sell the house, move, or downsize later?_

Output is an interactive HTML page with Plotly charts, tables, and
navigation tabs — viewable in any browser, no server required after
rendering.
