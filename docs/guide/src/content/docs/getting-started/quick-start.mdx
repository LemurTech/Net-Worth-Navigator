## Try the Sample Scenarios

Start the web editor from your Net Worth Navigator directory:

<Tabs>
  <TabItem label="Windows (PowerShell)">
    ```powershell
    .venv\Scripts\python.exe admin_app.py
    ```

    **Windows Firewall:** Windows Security may warn that *Windows Firewall has blocked some features of Python*. When this happens:

    - Click the **Allow access** notification
    - If a dialog opens, uncheck **Public networks** (keep Private networks checked)
    - Click **Allow access**

    This is safe — the editor only listens on your local machine.
  </TabItem>
  <TabItem label="Linux / macOS">
    ```bash
    .venv/bin/python admin_app.py
    ```
  </TabItem>
</Tabs>

Now open **[http://localhost:8010/setup](http://localhost:8010/setup)** in your browser. From the Setup Panel:

- Click **Render All** in the action bar at the bottom. This renders every sample scenario across all render modes — it may take a minute.
- Once complete, open **[http://localhost:8010/projection.html](http://localhost:8010/projection.html)** — your browser will show the interactive projection shell with charts, tables, and navigation tabs.
- Use the **scenario dropdown** at the top to switch between the four sample plans.

**Tip:** Press the **`?` button** in the toolbar to enable help mode — info icons appear on the KPI boxes and tab labels with plain-language explanations.

Both sample scenarios use **fictional data** (starting balances, income, life events) — no real financial information, no external accounts needed. They show you what a complete projection looks like with charts, cash flow tables, tax estimates, event timeline, and more.

## Create Your Own Scenario

Once you've explored the samples, create a household plan of your own.

### Web UI (Recommended)

- Open **[http://localhost:8010/setup](http://localhost:8010/setup)** (or restart the editor if needed)
- Click **New from Template** in the action bar at the bottom
- Choose **Single-Person** or **Couple / Two-Person**
- Enter a name (e.g. "Smith Retirement Plan") and a short slug (e.g. "smith")
- You're now in the Setup Panel for your new scenario:

  - **Metadata tab** — edit your plan name, description, and household members
  - **Accounts tab** — select your data source (Manual, Monarch, CSV) and enter starting balances
  - **Raw TOML tab** — directly edit the configuration file for advanced changes
  - Fill in your details, then click **Save** and **Render** (or just **Render**, which will validate and save your data before rendering)
  - Click **Open Projection** to see your results

**Prefer the command line?** See the [Command Line Basics](/Net-Worth-Navigator/getting-started/command-line-basics/) page for instructions on copying starter templates and running scenarios from the terminal.
