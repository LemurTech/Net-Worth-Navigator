This guide walks you through installing Net Worth Navigator on Windows, Linux, or macOS.

import { Tabs, TabItem } from '@astrojs/starlight/components';

## Prerequisites: Install Python

Net Worth Navigator requires **Python 3.11 or later**.

**New to Python?** Here's what to do:

- **Download Python** from [python.org/downloads](https://www.python.org/downloads/). Get the latest 3.11.x, 3.12.x, or 3.13.x release.
- **Install it.**

<Tabs>
  <TabItem label="Windows">
    1. **Check the box that says "Add Python to PATH"** during installation — this is critical. Without it, Windows won't know where to find Python when you type `python` in a terminal.
    2. Open **PowerShell** (right-click Start → Windows PowerShell) and verify:
       ```powershell
       python --version
       ```
       You should see something like `Python 3.12.4`.
  </TabItem>
  <TabItem label="Linux / macOS">
    Your system may already have Python installed. Open a terminal and check:

    ```bash
    python3 --version
    ```

    You should see something like `Python 3.12.4`. If you see `Python 3.11` or higher, you're good.

    **If Python is not installed or is too old:**

    - **Ubuntu / Debian:** `sudo apt install python3`
    - **Fedora:** `sudo dnf install python3`
    - **macOS:** Install via [python.org](https://www.python.org/downloads/) or `brew install python3`

    **On Linux:** You may also need to install the `python3-venv` package:

    ```bash
    sudo apt install python3-venv python3-pip
    sudo dnf install python3-virtualenv
    sudo pacman -S python-virtualenv
    ```
  </TabItem>
</Tabs>

**What's a "virtual environment"?** You'll see `.venv` in many commands below. A virtual environment is an isolated folder that keeps this project's dependencies separate from your other Python projects. You create it once and use it when you want to run the app.

## Get the Code

You need to download the Net Worth Navigator project files to your computer. First, decide where you want the project to live. A good choice is a folder where you keep your projects — for example:

<Tabs>
  <TabItem label="Windows">
    Create a folder like `C:\Users\yourname\Projects`. Open **PowerShell** and run:

    ```powershell
    cd $env:USERPROFILE
    mkdir Projects -Force
    cd Projects
    ```

    If you already have a folder you use for projects, `cd` into that instead.
  </TabItem>
  <TabItem label="Linux / macOS">
    Use a `Projects` folder in your home directory. Open a terminal and run:

    ```bash
    cd ~
    mkdir -p Projects
    cd Projects
    ```

    If you already keep projects somewhere else (like `~/code` or `~/dev`), use that instead.
  </TabItem>
</Tabs>

Now download the project. Pick one of the two methods below:

### Option A: Git Clone (recommended)

Git is the easiest way to get the code and keep it up to date later. If you don't have Git yet:

- **Windows:** Follow the [step-by-step guide to installing Git for Windows](/Net-Worth-Navigator/reference/installing-git-windows/).
- **Linux / macOS:** Git is usually pre-installed. Run `git --version` to check. If not, install it with your package manager (`sudo apt install git`, `brew install git`, etc.).

Once Git is ready, clone the repository:

<Tabs>
  <TabItem label="Windows (PowerShell)">
    ```powershell
    git clone https://github.com/LemurTech/Net-Worth-Navigator.git
    cd Net-Worth-Navigator
    ```
  </TabItem>
  <TabItem label="Linux / macOS">
    ```bash
    git clone https://github.com/LemurTech/Net-Worth-Navigator.git
    cd Net-Worth-Navigator
    ```
  </TabItem>
</Tabs>

### Option B: Download as ZIP

If you prefer not to install Git, you can download the project as a ZIP file:

1. Go to **[github.com/LemurTech/Net-Worth-Navigator](https://github.com/LemurTech/Net-Worth-Navigator)**
2. Click the green **Code** button, then **Download ZIP**
3. Extract the ZIP file into your `Projects` folder (or wherever you chose above)
4. Rename the extracted folder from `Net-Worth-Navigator-main` to `Net-Worth-Navigator`

Then open a terminal and `cd` into that folder:

<Tabs>
  <TabItem label="Windows (PowerShell)">
    ```powershell
    cd $env:USERPROFILE\Projects\Net-Worth-Navigator
    ```
  </TabItem>
  <TabItem label="Linux / macOS">
    ```bash
    cd ~/Projects/Net-Worth-Navigator
    ```
  </TabItem>
</Tabs>

**Tip:** Git makes updating the app much easier later on (a single `git pull` command). If you choose the ZIP method now, consider installing Git when you want to update.

## Set Up the Environment

<Tabs>
  <TabItem label="Windows (PowerShell)">
    ```powershell
    python -m venv .venv
    .venv\Scripts\python.exe -m pip install -r requirements.txt
    ```
  </TabItem>
  <TabItem label="Linux / macOS">
    ```bash
    python3 -m venv .venv
    .venv/bin/python -m pip install -r requirements.txt
    ```
  </TabItem>
</Tabs>

**Troubleshooting:** If `pip install` fails, make sure you're using the `.venv/bin/python` (or `.venv\Scripts\python.exe` on Windows) path — don't just run `pip install` directly, as that may use your system Python instead of the virtual environment.

## Verify the Installation

<Tabs>
  <TabItem label="Windows (PowerShell)">
    ```powershell
    .venv\Scripts\python.exe scripts/verify_install.py
    ```
  </TabItem>
  <TabItem label="Linux / macOS">
    ```bash
    .venv/bin/python scripts/verify_install.py
    ```
  </TabItem>
</Tabs>

This checks Python version, dependencies, and file structure. You should see `[OK] All checks passed!` at the end.
