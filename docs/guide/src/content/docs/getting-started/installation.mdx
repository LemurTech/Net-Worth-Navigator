This guide walks you through installing Net Worth Navigator on Windows, Linux, or macOS.

## Prerequisites: Install PythonSection titled "Prerequisites: Install Python"

Net Worth Navigator requires **Python 3.11 or later**.

**New to Python?** Here's what to do:

- **Download Python** from [python.org/downloads](https://www.python.org/downloads/). Get the latest 3.11.x, 3.12.x, or 3.13.x release.
- **Install it.**

1. **Windows:** **Check the box that says "Add Python to PATH"** during installation — this is critical.
- **Linux/macOS:** Your system may already have Python. Run `python3 --version` in a terminal to check.
- **Verify.** Open a terminal and run:

**What's a "virtual environment"?** You'll see `.venv` in many commands below. A virtual environment is an isolated folder that keeps this project's dependencies separate from your other Python projects. You create it once and use it when you want to run the app.

**On Linux:** You may also need to install the `python3-venv` package:

```bash
sudo apt install python3-venv python3-pip
sudo dnf install python3-virtualenv
sudo pacman -S python-virtualenv
```
## Get the CodeSection titled "Get the Code"
```bash
cd
git clone https://github.com/LemurTech/Net-Worth-Navigator.git
cd Net-Worth-Navigator
```

**Windows:** If you don't have Git installed, see the [step-by-step guide to installing Git for Windows](/Net-Worth-Navigator/reference/installing-git-windows/). You can also download the repository as a ZIP from the project page and extract it to a folder of your choice. (Git makes updating the app much easier later on!)

## Set Up the EnvironmentSection titled "Set Up the Environment"
Windows (PowerShell)Linux / macOS```powershell
python -m venv .venv
.venv\Scripts\python.exe -m pip install -r requirements.txt
``````bash
python3 -m venv .venv
.venv/bin/python -m pip install -r requirements.txt
```

**Troubleshooting:** If `pip install` fails, make sure you're using the `.venv/bin/python` (or `.venv\Scripts\python.exe` on Windows) path — don't just run `pip install` directly, as that may use your system Python instead of the virtual environment.

## Verify the InstallationSection titled "Verify the Installation"
Windows (PowerShell)Linux / macOS```powershell
.venv\Scripts\python.exe scripts/verify_install.py
``````bash
.venv/bin/python scripts/verify_install.py
```

This checks Python version, dependencies, and file structure. You should see `[OK] All checks passed!` at the end.
