- **Data Sources** — manual entry, CSV import, Monarch Money

## Who This Is For

Net Worth Navigator is a good fit if you:

- want a strategic, year-by-year planning model rather than a monthly budgeting app
- want to test scenario changes — retirement timing, spending, home sale plans, etc.
- are comfortable with simplified tax modeling (bracket-based federal, state tax, RMD)
- want to run everything locally with no ongoing subscription fees
- want interactive charts you can share with a partner or advisor

## Who This Is Not For

It is not a full-fidelity financial planning system. It is a poor fit if you need:

- Tax-return accuracy (NIIT, AMT, Medicare IRMAA, state-specific edge cases)
- Retirement withdrawal optimization (see [OWL](https://github.com/mdlacasse/Owl))
- Precise Social Security analysis across all benefit rules
- Estate planning or trust flows
- Monthly cash flow budgeting
