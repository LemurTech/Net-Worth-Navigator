Configuration Reference

Scenario settings live in scenarios/*.toml files. Each file is a complete self-contained financial plan.

All config keys use snake_case (lowercase with underscores). Section headers are also lowercase or snake_case.

Section titled “[simulation]”

real_dollar_basis — When true, all chart and table monetary figures are deflated to start-year purchasing power (for example, in 2026 dollars when start_year = 2026). The chart subtitle shows a “(in YYYY dollars)” tag and a value-basis note appears on the page. Default false (nominal future-dollar display).
