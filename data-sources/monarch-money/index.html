Monarch Money (Live Sync)

If you have a Monarch Money subscription, Net Worth Navigator can pull your live account balances on each run. This requires the Monarch MCP Server installed on your system.

Best for: active Monarch Money subscribers who want automatic balance updates.

Section titled “1. Install the Monarch MCP ServerSection titled “1. Install the Monarch MCP Server””

The Monarch MCP Server is a Python program that connects to Monarch’s API and provides your account data. It’s called directly by Net Worth Navigator as a subprocess — no AI agent, local LLM, or orchestrator is needed.

The MCP server runs on Windows, Linux, and macOS — anywhere Python runs.

Linux / macOSWindows```bash git clone https://github.com/robcerda/monarch-mcp-server.git /opt/monarch-mcp-server cd /opt/monarch-mcp-server uv run python login_setup.py # sends OTP to your email
